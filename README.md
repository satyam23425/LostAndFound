# 🔍 Reclaim — Lost & Found Platform

A modern, community-driven Lost & Found web application built with HTML, CSS, and JavaScript.

## 🌟 Features

- 🌙 **Dual Mode** — Seamless Light / Dark theme toggle
- 📦 **Live Item Feed** — Real-time grid of Lost & Found items
- 🔎 **Smart Search** — Instantly filter items by name, location, or category
- 📋 **Multi-Step Form** — Submit lost/found items with ID verification & geo-tagging
- 🔐 **Auth Modal** — Login / Sign-up with Google support
- ✅ **Verified Badges** — Trust system for verified community members
- 📍 **Geo-Tagging** — Auto-detect location when reporting an item
- 🔔 **Toast Notifications** — Real-time feedback on every action

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure |
| CSS3 | Styling, Glassmorphism, Animations |
| Vanilla JavaScript | Interactivity & Logic |

## 🚀 Getting Started

1. Clone the repository:
```bash
   git clone https://github.com/satyam23425/LostAndFound.git
```
2. Open `lost-and-found-app.html` in your browser
3. No installation required!
